💻 O que é um Computador?

Um computador é um dispositivo eletrônico projetado para realizar operações automáticas com base em dados fornecidos. Seu funcionamento é baseado em quatro grandes etapas::

🔁 Funcionalidades Principais

  1. 📥 Receber dados de entrada:
    • Dados originários de dispositivos: teclado, mouse, microfone, etc.
    • Podem ser dados brutos vindo de outros dispositivos ou vindos de comandos do usuário.

  2. 🧮 Processar os dados:
    • Operações: Realiza cálculos e lógica conforme instruções.
    • Responsável: CPU (Unidade Central de Processamento).

  3. 💾 Armazenar informações:
    • Temporário: Memória RAM para dados voláteis.
    • Permanente: HD/SSD para arquivos e sistema operacional.

  4. 📤 Fornecer uma saída (resultado):
    • Dispositivos: Monitor, impressora, caixas de som, etc.
    • Tipo de saída: Imagem, som, texto, gráfico, etc.

🔬 Dispositivo Mínimo que é um Computador

O menor dispositivo que ainda pode ser considerado um computador é o microcontrolador. Ele reúne os três elementos essenciais:

  • 🧠 Processador (CPU): executa instruções.
  • 📦 Memória: armazena dados e programas.
  • 🔌 Entradas/Saídas: permite comunicação com sensores, atuadores e usuários.

Dispositivos como o Arduino Uno ou ESP32 são exemplos de microcontroladores que podem controlar sistemas inteiros, mesmo com recursos bem limitados.
